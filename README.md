# Simple Actions Python

## Descripción

Hacer aciones en el ordenador muy facil!
Pulsar teclas, combinaciones, click especificos, mover raton i escribir.

#### Características

- Código fácil de leer
- Puedes tener el idioma español o ingles
- Open Source
